# React Account-Books

#### Stack

---

- React-router-dom
- json-server

#### Css

---

- css module
- UI glassmorphism 활용

> 구현기능

- 날짜 생성
- 회계부 CRUD기능
- 날짜 및 카드 전체 삭제
- 체크기능
- 지출 목록 이전/다음날짜로 이동
- 일별 합계 기능
- json-server를 활용해 db.json에 저장하여 rest api를 통해 fecth
